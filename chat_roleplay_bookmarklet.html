Chat Roleplay Helper

>What it does: When you are in ChatGPT using a roleplay prompt containing a 'Roleplay Scenario:' it will clean up the user interface by:

  1. Removing the login pane if you are not logged in, so you don't need to create an account.
  2. Roleplay prompts are really long and push the text down, this fixes that by hiding the first prompt in the page.
This will not work unless you Click it after entering ChatGPT via a Roleplay prompt similar to the ones you can make here.
