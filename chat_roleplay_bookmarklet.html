Chat Roleplay Helper

What it does:

When you are using a roleplay prompt in ChatGPT containing the text 'Roleplay Scenario:' it will clean up the user interface by:

  1. Removing the login pane, so you don't need to login or create an account.
  2. Roleplay prompts are really long and push the text down; this fixes that by hiding the first prompt on the page.
This will not work unless you click it after entering ChatGPT via a Roleplay prompt similar to the ones you can make here.
